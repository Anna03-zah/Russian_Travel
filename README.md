
## Описание проекта:
Адаптивный одностраничный сайт о туризме в России, созданный по макету Figma. 
Сайт адаптирован для просмотра на экранах различных устройств от 320 пикселей до широких экранов 1280 пикселей.

## Технологии:
БЭМ  
Flexbox  
Медиазапросы  
Подключение шрифтов
